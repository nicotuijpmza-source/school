# Nico's Assistent

Persoonlijke webapp voor rooster, WhatsApp, Gmail, Bunq en beleggingen.

## Starten

```bash
npm install
npm start
```

De app draait daarna op `http://localhost:3000`.

## Op telefoon zetten

Publiceer de app via een HTTPS-adres. Open dat adres daarna op je telefoon:

- iPhone: open in Safari, tik op delen, kies `Zet op beginscherm`.
- Android: open in Chrome, tik op het menu, kies `App installeren` of `Toevoegen aan startscherm`.

De app bevat een webmanifest, app-iconen en een service worker, zodat hij als webapp op je beginscherm kan staan.
